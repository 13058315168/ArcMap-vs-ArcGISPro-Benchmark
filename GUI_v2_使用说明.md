# ArcGIS Python2、3 与开源库性能对比测试工具 - GUI v2.0 使用说明

## 新功能特性

### 1. 多语言支持
- 支持中文/英文界面切换
- 设置自动保存，重启后保持选择

### 2. 图形化设置面板
通过菜单 **Settings → Test Settings** 打开设置面板：

#### 基本设置
- **Python路径配置**：通过文件浏览器选择Python 2.7和Python 3.x路径
- **环境验证**：一键验证Python环境和arcpy可用性
- **测试次数**：设置运行次数和预热次数
- **多进程选项**：启用/禁用多进程测试，设置worker数量
- **开源库测试**：启用/禁用GeoPandas/Rasterio测试

#### 数据规模设置
- 可视化编辑5种规模（tiny/small/standard/medium/large）的参数
- 每个参数可独立调整：
  - 渔网行列数
  - 随机点数量
  - 缓冲点数量
  - 叠加要素数量
  - 空间连接点数/面数
  - 字段计算记录数
  - 栅格大小
- 一键重置为默认值

#### 结果设置
- **选择性保存结果**：
  - 保存 Python 2.7 结果
  - 保存 Python 3.x 结果
  - 保存开源库结果
- **时间戳文件夹**：自动创建带日期时间的文件夹
- **结果保留策略**：设置自动清理天数（0=永久保留）

### 3. 进度显示
- **总进度条**：显示整体测试进度
- **当前测试**：显示正在运行的测试名称
- **预计剩余时间**（ETA）：基于已运行时间动态计算
- **单测试进度**：显示当前测试的内部进度

### 4. 快捷操作
- **一键打开临时文件夹**：快速访问测试数据和结果
- **日志操作**：清除/保存/复制日志
- **桌面快捷方式**：运行"创建桌面快捷方式.vbs"生成桌面图标

## 启动方式

### 方式1：无CMD窗口启动（推荐）
双击 `启动工具.vbs`
- 完全隐藏CMD窗口
- 使用pythonw运行

### 方式2：系统Python启动
双击 `启动工具_系统Python.vbs`
- 使用系统安装的pythonw
- 无需虚拟环境

### 方式3：命令行启动
```bash
python benchmark_gui_modern.py
```
- 适用于调试
- 可在控制台查看错误信息

### 方式4：创建桌面快捷方式
1. 双击 `创建桌面快捷方式.vbs`
2. 桌面上生成"ArcGIS Benchmark"快捷方式
3. 双击快捷方式即可启动

## 配置存储

设置自动保存在用户目录：
```
%USERPROFILE%\.arcgis_benchmark\config.json
```

包括：
- 界面语言
- Python路径
- 测试参数
- 数据规模配置
- 结果保存选项
- 窗口大小

## 测试结果存储

### 默认路径
```
C:\temp\arcgis_benchmark_data\YYYYMMDD_HHMMSS\<规模>/
```

### 临时数据路径
```
C:\temp\arcgis_benchmark_data\YYYYMMDD_HHMMSS\
```

### 可通过设置面板修改
- 自定义输出目录
- 禁用时间戳文件夹（不推荐）

## 常见问题

### Q: 启动后没有反应？
A: 检查系统是否安装了tkinter：
```python
python -c "import tkinter; print('OK')"
```

### Q: 无法找到Python路径？
A: 在设置面板中手动浏览选择python.exe路径，然后点击"验证"按钮确认。

### Q: 如何恢复默认设置？
A: Settings → Test Settings → 点击"Reset Default"按钮。

### Q: 测试运行时如何查看详细日志？
A: 使用命令行方式启动，可以在控制台看到完整的调试信息。

## 与旧版GUI的区别

| 功能 | 旧版GUI | GUI v2.0 |
|------|---------|----------|
| 参数修改 | 需编辑settings.py | 图形化设置面板 |
| 多语言 | 仅中文 | 中英文切换 |
| 进度显示 | 简单文本 | 进度条+ETA |
| Python路径 | 自动检测 | 可视化选择+验证 |
| 结果控制 | 全部保存 | 选择性保存 |
| 文件夹组织 | 固定路径 | 时间戳文件夹 |
| CMD窗口 | 显示 | 可选隐藏 |
| 数据规模调整 | 修改代码 | GUI直接编辑 |
